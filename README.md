# Hugo Blog

## Hugo 설치 하기

- [Hugo Install and Use Themes](https://gohugo.io/themes/installing-and-using-themes/)

### Ubuntu
```bash
sudo apt install hugo
```
### 새글 쓰기

```bash
hugo new post/[글제목].md
```


## 서버 실행하기

```bash
hugo server
```
